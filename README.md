# Plugin-Spam

Spam Troll — плагин для троллинга в чатах, который автоматически отправляет жертве поток оскорбительных и провокационных сообщений. 

Поддерживает как встроенные грубые фразы, так и пользовательские тексты. Управляется командами .troll (запуск) и .stoptroll (остановка). 

Позволяет настраивать задержку между сообщениями, их количество, а также выбирать только свои фразы или комбинировать со стандартными. Включает удобный интерфейс для добавления, редактирования и удаления пользовательских оскорблений. 

Работает в фоновом режиме, не блокируя интерфейc!
